# Error Codes

| Error Code | Description              |
| ---------- | ------------------------ |
| 2000       | Success                  |
| 2001       | Transaction created      |
| 2002       | Callback accepted        |
| 4000       | Unknown error            |
| 4001       | Too many request         |
| 4002       | Missing signature header |
| 4003       | Invalid HMAC signature   |
| 4004       | Invalid token            |
| 4005       | Validation failed        |
| 4006       | Transaction not found    |
| 5001       | Payment failed           |
